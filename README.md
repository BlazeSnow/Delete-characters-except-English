# 写在最前

1. 本程序用于删除除了英文和空格的字符
2. 第一次使用请生成新文件，将需处理内容复制进新文件后处理文件
3. 程序将删除英文括号及括号内内容
4. 程序将删除表示单词类型的缩写

|   各   |   种   |   缩   |   写   |
|:-----:|:-----:|:-----:|:-----:|
|  n.   | pron. | art.  | num.  |
| adj.  | adv.  |  v.   | conj. |
| prep. | int.  | abbr. |  vt.  |
|  vi.  |       |       |       |
